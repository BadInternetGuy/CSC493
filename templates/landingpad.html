GameChat

A free, easy to use website that allows gamers to connect with gamers. Select from our listed games, select a discussion topic, and get chatting!

Click here to get started

Built with Bootstrap 4 and Flask

Welcome to GameChat. This website seeks to aid those who want a safe place to discuss whatever game they wish to. This is a work in progress.

Supported Games

GameChat is separated into boards based on different games. At the moment, only a few games are suported, but several more are planned. Each game has their own forum, divided into its own discussion topics. The following games are planned:

  • Anthem
  • Destiny
  • League of Legends
  • Hollow Knight
  • Magic: The Gathering

Discussions

Every game board is separated into different discussion topics. This ensures that you can always talk about what you want to talk about.

Topics

Every board is different, but all will have the following topics in common:

  • General Discussion
  • Lore
  • Player Feedback
  • Players Helping Players
